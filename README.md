# DS Audit Plugin

A Cowork plugin for design system auditing, updating, and handover. Built for design teams using Claude Code and Notion for DS documentation.

## Requirements
- Completed Advanced Guide 1/5 (DS Extraction)
- Completed Advanced Guide 2/5 (Notion Documentation)
- Figma Pro plan or higher
- Claude Pro, Max, Team, or Enterprise plan

## Installation
Install directly from this repository via the Cowork plugin marketplace.

## Commands
- /ds-audit-plugin:setup-ds -- one-time guided setup
- /ds-audit-plugin:audit-ds -- full connected audit (coming soon)
- /ds-audit-plugin:update-ds -- DS change detection (coming soon)
- /ds-audit-plugin:handover-ds -- handover readiness check (coming soon)
